Garage Sally

A responsive web app that allows people to post, search, and share local garage sales, combined with a check-in feature. It's like in-person ebay combined with the functionality of a check-in app.
Say good-bye to driving around endlessly hunting for signs or fliers. And with the check-in functionality, you can see how popular a garage sale is, so you can know before you go.

About this project

  • My role as Back-End Engineer
    • Implemented RESTful back-end APIs using Mongoose to make calls to MongoDB
    • Configured a Node.js server using Express.js to link the front-end ReactJS client
    • Created and maintained back-end Mongoose models for validating data to store in MondoDB
    • Incorporated a file/image upload feature using a native JavaScript front-end converter to map files to Mongoose models
